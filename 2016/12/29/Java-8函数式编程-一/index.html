Java 8函数式编程(一)

发表于 2016-12-29 |

概述

第一章主要介绍了为什么Java8需要引入Lambda以及为整本书定调, 作者不会在诸如lambda是否符合函数式编程风格等问题而纠结.

第二章介绍了Lambda的基本特性, 包括常见的几种表达式, 函数接口, 类型判断等.

重要摘记

  • 为了能够使代码在多核CPU上高效运行, 编写这类处理批处理数据的并行类库, 需要在语言层面上修改闲鱼的Java:增加Lambda表达式.
  • 面向对象编程是对数据进行抽象，而函数式编程是对行为进行抽象.
  • 函数式编程的核心思想是: 在思考问题时, 使用不可变值和函数, 函数对一个值进行处理, 映射成另一个值.
  • 我们关心的是如何写出“好”代码, 而不是符合函数式编程风格的代码.
